# SQL 访问和处理数据库的标准的计算机语言
## SQL语言的分类(基于功能划分为4类)

* 数据定义：DDL (Data definition Language)
  * 库的创建删除、表的创建删除等
* 数据操纵：DML (Data Manipulation language)
  * 新增数据、删除数据、修改数据等
* 数据口精致：DCL (Data control Language)
  * 新增用户、删除用户、密码修改、权限管理等
* 数据查询：DQL (Data Query Language)
  * 基于需求查询和计算数据
## SQL 语法特征
1. 大小写不敏感
2. 可以多行写入,但要以`;`来结束
3. 支持注释:
   * 单行注释: `-- 注释内容`
   * 单行注释: `# 注释内容`
   * 多行注释: `/* 注释内容 */`
