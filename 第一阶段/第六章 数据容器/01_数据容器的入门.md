
# 1、什么是数据容器？
一种可以存储多个元素的Python数据类型

# 2、Python有哪些数据容器？
list（列表）、tuple（元祖）、str（字符串）、set（集合）、dict（字典）
他们各有特点，但都满足可容纳多个元素的特性

# 3、常见的容器定义符号汇总
## 定义元组
```bazaar
# 定义元组
t1 = (1, "hello", True)
```
```bazaar
# 定义一个空元组
t2 = ()
t3 = tuple()
```
## 定义列表
```bazaar
# 定义一个列表list
my_list = ["itheima","itcast","python"]
```
```bazaar
# 定义一个嵌套的列表
my_list = [[1, 2, 3],[4, 5, 6]]
```

## 定义字符串
```bazaar
# 定义一个字符串
my_str = "itheima and itcast"
```
## 定义集合
```bazaar
# 定义集合
my_set = {"传智教育", "黑马程序员", "itheima", "传智教育", "黑马程序员", "itheima", "传智教育", "黑马程序员", "itheima",}
```
